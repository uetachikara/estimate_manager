# README
# Estimate Manager（見積・案件管理アプリ）

Rails を用いて作成した、**取引先・案件・見積を一元管理する業務向けWebアプリケーション**です。  
実務を想定し、「ユーザー操作で一連の業務フローが完結すること」を重視して設計・実装しています。

---

## アプリ概要

- 取引先（Client）管理
- 案件（Project）管理
- 見積（Quote）管理
- ステータス管理（enum + 日本語表示）
- ユーザー認証（Devise）
- system spec による E2E テスト

---

## 使用技術

- Ruby 3.3
- Ruby on Rails 8.1
- MySQL
- Bootstrap 5
- Devise
- RSpec / Capybara（system spec）

---

## 主な機能

### 認証機能
- ログイン / ログアウト
- 新規ユーザー登録
- パスワード再設定
- ログイン画面・再設定画面は業務アプリを意識したUIにカスタマイズ

### 取引先管理
- 取引先の登録 / 編集 / 削除
- 一覧で案件数を表示（N+1 を避けた実装）

### 案件管理
- 案件の登録 / 編集
- ステータス管理（リード / 進行中 / 完了）
- ステータスは **色付きバッジ + 日本語表示**

### 見積管理
- 見積の登録 / 編集
- 見積ステータス管理（下書き / 送付済み / 受注 / 失注）
- 金額表示を業務向けに整形

---

## 画面イメージ

- ログイン画面（カードUI）
- 取引先一覧（案件数表示）
- 案件一覧（ステータスバッジ）
- 詳細画面（カード + 定義リスト形式）

※ スクリーンショットは後日追加予定

## テスト

### System Spec（E2E）

ログインから見積作成までの一連の業務フローを  
**system spec（Capybara）で検証**しています。

テスト内容：

ユーザーがログインできる

取引先を作成できる

案件を作成できる

見積を作成できる

セットアップ手順
git clone https://github.com/uetachikara/estimate_manager.git
cd estimate_manager

bundle install
rails db:create db:migrate
bin/dev


ブラウザで以下にアクセス：

http://localhost:3000
